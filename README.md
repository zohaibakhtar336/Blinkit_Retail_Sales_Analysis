# 🛒 Blinkit Retail Sales Analysis (Power BI)

## 📌 Project Overview

This project presents an end-to-end **Power BI dashboard solution** designed to analyze Blinkit’s retail sales performance.

It focuses on extracting actionable insights from raw data to understand **sales trends, customer behavior, and outlet performance**, enabling better business decision-making in a quick-commerce environment.

---

## 🎯 Key Highlights

* 📊 Designed an interactive Power BI dashboard with dynamic slicers
* 📈 Performed **sales trend analysis** across multiple dimensions
* 🏪 Analyzed **outlet performance** based on size, type, and location
* 🛍️ Explored **product category contribution** to total sales
* ⭐ Evaluated customer behavior using ratings and item visibility
* ⚡ Built KPI-driven reporting for quick business insights

---

## 📊 Key Metrics

* 💰 **Total Sales:** $1.2M
* 📊 **Average Sales:** $141
* ⭐ **Average Rating:** 3.9
* 📦 **Total Items:** 8523

---

## 📈 Detailed Analysis

### 🔹 Sales by Item Type

* Identified top-performing categories such as Fruits, Snacks, and Dairy
* Compared contribution of each category to overall revenue

### 🔹 Outlet Performance Analysis

* Evaluated performance across **Small, Medium, and High-size outlets**
* Compared outlet types based on sales, ratings, and visibility

### 🔹 Location-Based Insights

* Analyzed sales across **Tier 1, Tier 2, and Tier 3 cities**
* Observed stronger performance in Tier 1 locations

### 🔹 Product Insights

* Compared **Low Fat vs Regular products**
* Identified trends in customer preference and demand
